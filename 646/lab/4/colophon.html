Colophon

First, I went to the Bootstrap Site and downloaded Bootstrap to my lab/4 directory.

Go to the Basic Template Section to copy the start.html file contents.

From http://getbootstrap.com/components/#navbar I chose a Default navigation Bar to include in my start.html file.

I changed my navbar links to Paint, Walls & Plumbing.

Add the JumboTron to the index.html file.

Make the Paint link into a dropdown linking to other pages.

I didn't like how the dropdown behaved so I stole a concept from a stackoverflow question answered usingjsfiddle.

That being said, I do have two new classes I've added: caret-before and caret-after which only adjust padding really for the caret.
